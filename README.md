# Modern Landing Page

A responsive, modern landing page template built with HTML, CSS, and JavaScript.

## Features

- Responsive design for all devices
- Modern UI with smooth animations
- Mobile-friendly navigation
- Contact form with validation
- Clean and maintainable code

## Getting Started

1. Open `index.html` in your browser
2. Customize the content to match your needs
3. Update colors and styles in `styles.css`

## Project Structure

```
├── index.html    # Main HTML file
├── styles.css    # CSS styles
├── script.js     # JavaScript functionality
└── README.md     # This file
```

## Customization

- **Colors**: Modify CSS variables in `styles.css`
- **Content**: Edit HTML in `index.html`
- **Functionality**: Add JavaScript in `script.js`
